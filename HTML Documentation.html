    • Sceleton of HTML

    All webpages begin with "!DOCTYPE html"
    After that you have a "html and /html" with everything in it
    The first thing you will want to do after that is a "head and /head" tag
    Inside, you will need "meta charset="utf-8"" as a whole tag
    This is where you will also need to put your "style and /style" with your CSS inside
    And lastly you will need "body and /body" tags to hold everything that the veiwer sees

    Thanks to LHS Digital Media for this image of an HTML skeleton

    Text tags

    The main tag you would use to make headings for paragraphs is the "h# and /h#" tags. These headings will get smaller as the number after h goes up
    After that you have the "p and /p" tags to make the paragraphs that go below your headers
    To make special effects for these text use the tags "em" to make slanted writing or "strong" to make bold writing

    CSS

    The way you write CSS is by going to (or making) the "style" tag in the "head" tag
    CSS colors and formats anything you select, you select it by giving it an ID or a class
    In order to color/ format you must:

    h1{
    color: blue
    }
    or
    #id{
    color: red
    }
    or
    .class{
    color: green
    }

    Some CSS tags are:

  • color
  • font-family
  • font-size
  • font-weight
